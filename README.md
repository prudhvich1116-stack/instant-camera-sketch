# Instant Camera Face Sketch Generator

This project converts real-time webcam images into pencil sketch style images using Python and OpenCV.

## Technologies
- Python
- OpenCV
- NumPy

## Features
- Real-time face detection
- Pencil sketch image generation
- Image capture and saving
- Keyboard controls for capture and sharing

## Usage
Press:
C – Capture image  
S – Share image  
Q – Exit application
